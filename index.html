Architect and Build an End-to-End AWS Portfolio Web-App (AWS)

In this project, I made use of Python and AWS to create an intelligent data science portfolio website. At a high-level, the website will be hosted in a serverless manner on AWS; showcasing my amazing data engineering projects.

The investigation of Water Works In The Rand Water Supply System

Here, we designed a Web application for providing real-time insights on the different performance metrics for the different water treatment plants and catchment areas for Rand Water in South Africa. We then deployed using knowledge of time series, deep learning models for making future predictions with regards to the performance and compliance of these treatment plants with W.H.O. Standards. In the 6th Minute of the video presentation, I described the thought process behind the architecture, walking us through the steps taken for ther different ELT pipelines up until the point of deployment. We worked as a team of 2 Data engineers and 10 Data scientists.

COVID-19 DATA INFRASTRUCTURE

This project focuses on building an data infrastructure leveraging on AWS services such as crawler, Redshift, AWS glue, AWS Athena and many more. It is a simple data engineering architecture which is deployed using entirely python.

Uber Data Analytics Using Mage GCP infrastructure

Here, we designed a data engineering infrastructure capable of carrying out simple ETL processes on our data, leveraging Google Cloud Platform as the cloud vendor. We used Mage, an open-source tool for building ETL pipelines, for this. With Mage, We designed a pipeline that extracted the data from the storage, transformed this data, and built our data model comprising fact and dimension tables. We then loaded the transformed data into our warehouse (BigQuery), where we created a custom table and visualized it on an interactive dashboard using Looker Studio.

NFT Art Price Prediction App

Given the date of an auction, the artist, the image of the NFT art piece, historical prices and various other features, this model will give an estimation of the auction price for that particular NFT art piece.

Hospital Inventory Monitor

This piece of software takes in various financial, operational, strategic and process metrics and returns an estimate of the hospitals inventory. This model also flags anomalous events like when certain drug go out of stock and tries to forecast if performance will degrade over time and what the preventative measures might be to reduce/ nullify performance degradation.
